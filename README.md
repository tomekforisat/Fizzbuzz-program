# Fizzbuzz-program
This program will allow for any number between 1-100 to be output as the number, fizz, buzz, or fizzbuzz depening on whether it is a multiple of 3, 5, or 15.  

def Fizzbuzz(number):
    if number % 3 == 0 and number % 5 ==0:
        print ("fizzbuzz")
    else:
        if number % 3 == 0:
            print ("fizz")
        else: 
            if number % 5 ==0: 
                print("buzz")
            else:
                print(number)
